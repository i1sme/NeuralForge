# M14 LayerNorm (A2 second brick) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `StdOp::LayerNorm` end-to-end on both arm64 and x86_64 profiles as A2 second brick (after M13's `add`). Single composite op with internal multi-pass codegen, mirroring the Softmax-as-one-node precedent. Native hardware sqrt — no libm dependency added.

**Architecture:**
- **Foundation (Task 1):** new `StdOp::LayerNorm` variant + `Signature` (`positional: []`, `named: [{ name: "affine", ty: Symbol, required: false }]`); `infer_output_shape` is identity with rank ≥ 2 design constraint (mirroring Softmax — not a math limit); `ParamKind` extended with `LayerNormScale` and `LayerNormBias` (γ before β — contract). Both profiles get stub `emit_layernorm` returning `LowerError::UnsupportedOp`, `walk_model` dispatch arm, `classify_op` arm, and `ParamSlot` allocation logic gated on `layernorm_has_affine(attrs)` (new helper parallel to existing `linear_has_bias`, mirror naming convention).
- **arm64 emit (Task 2):** 3-pass per row (mean → variance + inv_std → normalize + optional affine). Native `fsqrt s, s`. Leaf function (no FFI). Scratch in x9–x16 + s0–s7. AAPCS64-safe — `s_b` reuses `s2` (after `s_inv_d` consumption) instead of `s8` (which is callee-saved per AAPCS64 §6.1.2). Constants in `.rodata` (Mach-O `__TEXT,__const`), partial reload per row.
- **x86_64 emit (Task 3):** same 3-pass shape in AT&T syntax. Native `sqrtss`. Op-local `pushq %r12`/`pushq %r13` at start of emit body and matching `popq` at end — **only when has_affine == true**. `compute_callee_saved` is **NOT** modified (M13 pre-Task-5 arm64 precedent for op-local save/restore). Stack alignment invariant noted for M-future Softmax+LayerNorm-with-affine co-location.
- **Fixtures + FFI (Task 4):** three positive (`layernorm_no_affine.nfl` N=1, `layernorm_affine.nfl` N=1, `pre_ln_block.nfl` N=2 — last triggers Plan 1's LH-1 closure end-to-end via bit-exact compare); one negative (`layernorm_rank_too_low.nfl` — IR-level reject). Reference Rust impl uses sequential `for` reduction (LLVM does not reorder f32 adds without `-ffast-math`) and pre-computed `inv_d = 1.0/D` (matches emitter's compile-time constant bit-exactly).
- **Docs (Task 5):** DEVLOG, PROJECT_SPEC §"Current Status" + Strategic Roadmap A2 annotation, CLAUDE.md tree + status, `docs/language_reference/{grammar,uir}.md`, `docs/profile_guide/{arm64,x86_64}.md`. The x86_64 profile guide cross-references Plan 1's LH cleanup commit.

**Tech Stack:** Rust 2024 workspace; macOS arm64 (host) + Linux x86_64 (CI); `cc` + `libloading` for FFI integration tests.

```rust
// SPDX-License-Identifier: Apache-2.0

//! LayerNorm codegen — arm64 (M14).
//!
//! Per-row 3-pass: mean (Σ x_i, /D) → variance + inv_std (Σ (x_i−μ)²,
//! /D, +ε, fsqrt, 1/sqrt) → normalize + optional affine
//! ((x_i−μ) · inv_std [· γ_i + β_i]).
//!
//! Native `fsqrt s, s` — no libm call. Leaf function (no FFI save/restore).
//!
//! Register plan (M14 spec §7, finalized):
//!   x6   = bound scratch (clobbered every emit_imm32 call) — pure scratch
//!   x9   = x_in  (per-row input ptr;  recomputed at top of each row)
//!   x10  = x_out (per-row output ptr; recomputed at top of each row)
//!   x11  = x_j (inner counter)
//!   x12  = x_i (outer counter)
//!   x13  = x_gamma (γ base ptr — affine only)
//!   x14  = x_beta  (β base ptr — affine only)
//!   x15  = x_const_pool (.rodata constants base — set ONCE before outer loop;
//!                        lives entire function; NOT clobbered by any inner emit_imm32
//!                        because all bound loads target x6, not x15)
//!   x16  = src_base (set ONCE via materialise_ptr; lives entire function)
//!   x17  = dst_base (set ONCE via materialise_ptr; lives entire function)
//!   s0   = s_acc (accumulator); reused as s_var at end of Pass 2 (4-instruction lifetime)
//!   s1   = s_mean (live Pass 2 + Pass 3)
//!   s2   = s_inv_d (1/D constant); REUSED as s_b in Pass 3 affine; reload from x15 pool
//!          at end of each row's Pass 3 (strategy (b) per spec §11.8 — only s2 reloads)
//!   s3   = s_eps (1e-5; live across outer batch loop)
//!   s4   = s_one (1.0;  live across outer batch loop)
//!   s5   = s_inv_std (Q4 constraint — held through Pass 3, NOT recomputed in hot loop)
//!   s6   = s_t (per-element temp)
//!   s7   = s_g (γ_j load — affine only)
//!
//! All scratch in caller-saved range (x6, x9–x17 + s0–s7). v8–v15 (= s8–s15)
//! INTENTIONALLY AVOIDED — AAPCS64 §6.1.2 makes them callee-saved (lower
//! 64 bits); writing them in a leaf function without stp/ldp save would
//! silently corrupt caller's v8–v15. Reusing s2 (s_inv_d) for s_b in
//! Pass 3 keeps the float register footprint within s0–s7.
//!
//! Note on x16/x17: AAPCS64 marks these as "intra-procedure call scratch"
//! (IP0/IP1) — caller-saved, used by linker stubs across `bl` calls.
//! emit_layernorm has NO `bl` calls (leaf function), so x16/x17 are
//! effectively free for op use.
//!
//! Constants in .rodata pool — addressable via `adrp + add` once at
//! function start, pool itself emitted at function bottom or via shared
//! per-file pool. Mach-O directive `__TEXT,__const` on macOS arm64.
//! See implementation notes after this code block for fallback if
//! per-function pool emission is not yet supported by `asm.rs`.
//!
//! M13 ABI register-conflict lesson generalized: all scratch lives in
//! non-INPUT_REGS scope from the start. LH-class bugs structurally
//! impossible in fresh emit_layernorm code at any N=1..4.

use crate::abi::AbiContext;
use crate::asm::emit_imm32;
use crate::buffer::BufferLoc;
use compiler::ast::Span;
use profile_api::LowerError;

#[allow(clippy::too_many_arguments)]
pub fn emit_layernorm(
    abi: &AbiContext,
    b: u64,
    d: u64,
    model_idx: usize,
    layernorm_idx: usize,
    src_loc: BufferLoc,
    dst_loc: BufferLoc,
    gamma_offset: Option<usize>,
    beta_offset: Option<usize>,
    _node_span: Span,
) -> Result<String, LowerError> {
    let lid = format!("{model_idx}_{layernorm_idx}");
    let has_affine = gamma_offset.is_some();
    debug_assert_eq!(
        has_affine,
        beta_offset.is_some(),
        "γ and β offsets must be both Some or both None — order contract enforced upstream"
    );

    let mut s = String::new();
    s.push_str(&format!(
        "    ; layernorm (3-pass, {}affine): input [{b},{d}] -> output [{b},{d}]\n",
        if has_affine { "with " } else { "no-" },
    ));

    // Materialise base pointers ONCE before the outer loop. x16/x17 are
    // safe in this leaf function (no `bl` to trigger linker stubs).
    abi.materialise_ptr(src_loc, "x16", &mut s);
    abi.materialise_ptr(dst_loc, "x17", &mut s);

    // Affine: materialise γ and β base pointers from params_reg into x13/x14.
    // x6 used as transient byte-offset scratch for non-zero offsets.
    if has_affine {
        let g_off = gamma_offset.unwrap();
        let b_off = beta_offset.unwrap();
        let params_reg = abi.params_reg();
        if g_off == 0 {
            s.push_str(&format!("    mov     x13, {}\n", params_reg));
        } else {
            s.push_str(&emit_imm32("x6", g_off * 4));
            s.push_str(&format!("    add     x13, {}, x6\n", params_reg));
        }
        if b_off == 0 {
            s.push_str(&format!("    mov     x14, {}\n", params_reg));
        } else {
            s.push_str(&emit_imm32("x6", b_off * 4));
            s.push_str(&format!("    add     x14, {}, x6\n", params_reg));
        }
    }

    // Pre-loop hoisted constants. Layout in .rodata pool:
    //   offset 0: .float  1.0 / D   → s2 (s_inv_d)
    //   offset 4: .float  1e-5      → s3 (s_eps)
    //   offset 8: .float  1.0       → s4 (s_one)
    //
    // x15 is set to the pool base ONCE here and never clobbered by any
    // bound load below (bound loads target x6). This makes the affine-path
    // s2 reload at end of each row's Pass 3 a single `ldr s2, [x15, #0]`
    // — no need to re-establish x15 via adrp/add.
    let inv_d_bits = (1.0_f32 / d as f32).to_bits();
    let eps_bits = 1e-5_f32.to_bits();
    let one_bits = 1.0_f32.to_bits();
    s.push_str(&format!("    ; constants: inv_d=0x{:08x}, eps=0x{:08x}, one=0x{:08x}\n",
                        inv_d_bits, eps_bits, one_bits));
    s.push_str(&format!("    adrp    x15, .Lln_consts_{lid}@PAGE\n"));
    s.push_str(&format!("    add     x15, x15, .Lln_consts_{lid}@PAGEOFF\n"));
    s.push_str("    ldr     s2, [x15, #0]\n");   // s_inv_d
    s.push_str("    ldr     s3, [x15, #4]\n");   // s_eps
    s.push_str("    ldr     s4, [x15, #8]\n");   // s_one

    // === outer row loop: i in 0..b ===
    s.push_str("    mov     x12, #0\n");
    s.push_str(&format!(".Lln_row_{lid}:\n"));
    s.push_str(&emit_imm32("x6", b as usize));
    s.push_str("    cmp     x12, x6\n");
    s.push_str(&format!("    b.ge    .Lln_row_end_{lid}\n"));

    // Compute per-row pointers: x9 = src_base + i*d*4; x10 = dst_base + i*d*4.
    // x6 holds the byte stride (i * d * 4) temporarily.
    s.push_str(&emit_imm32("x6", (d as usize) * 4));
    s.push_str("    mul     x6, x12, x6\n");      // x6 = i * d * 4 (byte offset)
    s.push_str("    add     x9,  x16, x6\n");
    s.push_str("    add     x10, x17, x6\n");

    // === Pass 1: μ = (1/D) · Σ x_j ===
    s.push_str("    fmov    s0, wzr\n");          // s_acc = 0
    s.push_str("    mov     x11, #0\n");          // x_j = 0
    s.push_str(&format!(".Lln_p1_{lid}:\n"));
    s.push_str(&emit_imm32("x6", d as usize));
    s.push_str("    cmp     x11, x6\n");
    s.push_str(&format!("    b.ge    .Lln_p1_end_{lid}\n"));
    s.push_str("    ldr     s6, [x9, x11, lsl #2]\n");
    s.push_str("    fadd    s0, s0, s6\n");
    s.push_str("    add     x11, x11, #1\n");
    s.push_str(&format!("    b       .Lln_p1_{lid}\n"));
    s.push_str(&format!(".Lln_p1_end_{lid}:\n"));
    s.push_str("    fmul    s1, s0, s2\n");       // s_mean = s_acc * s_inv_d

    // === Pass 2: σ² = (1/D) · Σ (x_j − μ)²; inv_std = 1/sqrt(σ² + ε) ===
    s.push_str("    fmov    s0, wzr\n");          // s_acc = 0
    s.push_str("    mov     x11, #0\n");
    s.push_str(&format!(".Lln_p2_{lid}:\n"));
    s.push_str(&emit_imm32("x6", d as usize));
    s.push_str("    cmp     x11, x6\n");
    s.push_str(&format!("    b.ge    .Lln_p2_end_{lid}\n"));
    s.push_str("    ldr     s6, [x9, x11, lsl #2]\n");
    s.push_str("    fsub    s6, s6, s1\n");
    s.push_str("    fmul    s6, s6, s6\n");
    s.push_str("    fadd    s0, s0, s6\n");
    s.push_str("    add     x11, x11, #1\n");
    s.push_str(&format!("    b       .Lln_p2_{lid}\n"));
    s.push_str(&format!(".Lln_p2_end_{lid}:\n"));
    // s_var lives in s0 — s_acc is dead after the next fmul reads it.
    s.push_str("    fmul    s0, s0, s2\n");       // s_var = s_acc * s_inv_d
    s.push_str("    fadd    s0, s0, s3\n");       // s_var += s_eps
    s.push_str("    fsqrt   s0, s0\n");           // s_var = sqrt(σ² + ε)
    s.push_str("    fdiv    s5, s4, s0\n");       // s_inv_std = s_one / s_var
    //         ^^^^^^^^^^^^^^^^^^^^^^^^^^^ Q4 constraint: ONE fdiv per row,
    //         OUTSIDE Pass 3. Pass 3 hot loop uses fmul s_t, s_t, s_inv_std.

    // === Pass 3: y_j = (x_j − μ) · inv_std [· γ_j + β_j] ===
    s.push_str("    mov     x11, #0\n");
    s.push_str(&format!(".Lln_p3_{lid}:\n"));
    s.push_str(&emit_imm32("x6", d as usize));
    s.push_str("    cmp     x11, x6\n");
    s.push_str(&format!("    b.ge    .Lln_p3_end_{lid}\n"));
    s.push_str("    ldr     s6, [x9, x11, lsl #2]\n");
    s.push_str("    fsub    s6, s6, s1\n");
    s.push_str("    fmul    s6, s6, s5\n");       // s_t *= s_inv_std (NOT fdiv — Q4)
    if has_affine {
        s.push_str("    ldr     s7, [x13, x11, lsl #2]\n");  // γ_j → s_g
        s.push_str("    fmul    s6, s6, s7\n");
        s.push_str("    ldr     s2, [x14, x11, lsl #2]\n");  // β_j → s_b (REUSES s2)
        s.push_str("    fadd    s6, s6, s2\n");
    }
    s.push_str("    str     s6, [x10, x11, lsl #2]\n");
    s.push_str("    add     x11, x11, #1\n");
    s.push_str(&format!("    b       .Lln_p3_{lid}\n"));
    s.push_str(&format!(".Lln_p3_end_{lid}:\n"));

    // Reload s_inv_d (s2) for next row's Pass 1 — only when affine
    // (Pass 3 affine clobbered s2 with the β load). For no-affine, s2
    // still holds the original constant. x15 still points to the pool
    // base (never clobbered above — bound loads use x6, not x15).
    if has_affine {
        s.push_str("    ldr     s2, [x15, #0]\n");
    }

    s.push_str("    add     x12, x12, #1\n");
    s.push_str(&format!("    b       .Lln_row_{lid}\n"));
    s.push_str(&format!(".Lln_row_end_{lid}:\n"));

    // Per-function .rodata constants pool — emitted at the bottom of the
    // function body or as part of a shared per-file pool. See implementation
    // notes after this code block for asm.rs integration guidance.
    s.push_str(&format!("    ; .Lln_consts_{lid}: 1/D=0x{inv_d_bits:08x}, eps=0x{eps_bits:08x}, one=0x{one_bits:08x}\n"));
    s.push_str(&format!("    ; (emit pool via existing asm.rs convention; if no per-function pool support yet, see fallback below)\n"));

    Ok(s)
}
```

**Implementation notes for the engineer:**

1. **Constant pool emission integration with `asm.rs`.** The `adrp .Lln_consts_{lid}@PAGE` + `add ... @PAGEOFF` block above assumes a per-function `.rodata` pool labeled `.Lln_consts_<lid>` is emitted somewhere reachable. The exact emission mechanism depends on what `asm.rs::format_function_epilogue` (or a sibling helper) currently supports:

   - **If a per-function pool helper exists** (look for any `.section __TEXT,__const` emission in `profiles/arm64/src/asm.rs` or a `pool` module): use it. Pass the three constants to the helper at the end of `emit_layernorm`.

   - **If only a per-file pool exists or no pool support at all**: fall back to inline immediate materialization for the constants. Replace the `adrp/add/ldr` block with `movz`/`movk` immediates per constant, hoisted before the outer loop:

     ```rust
     // Inline materialization fallback (no .rodata pool needed):
     // For each constant: movz w_lo, #(bits & 0xFFFF); movk w_lo, #((bits >> 16) & 0xFFFF), lsl #16; fmov s_target, w_lo.
     // ~3 instructions × 3 constants = 9 instructions, hoisted once before outer loop.
     // Reload of s_inv_d at end of Pass 3 affine becomes the same 3-instruction sequence
     // re-emitted (12 extra instructions per row × B rows — still negligible vs O(D) per-row work).
     // Use w0 as scratch (caller-saved scratch GPR for materialization, distinct from x6).
     ```

     The inline form is what `emit_softmax`'s `-inf` constant materialization uses (`profiles/arm64/src/ops/softmax.rs:81-83` — `movz`/`movk` pair). For three f32 constants the inline form is uglier than a pool but works without changes to `asm.rs`. **Recommended: try per-function pool first; fall back to inline if pool integration adds significant `asm.rs` surface.**

2. **Mach-O section directive vs ELF.** If a per-function pool IS emitted, the section directive on macOS arm64 is `.section __TEXT,__const`, NOT `.section .rodata, "a"` (the latter is ELF). Match the existing arm64 convention — verify by grepping `.section` through `profiles/arm64/src/asm.rs` and existing op emitters.

3. **Stable register allocation.** The plan above is internally consistent — x6 is the SOLE bound register (clobbered by every `emit_imm32` call), x15 holds the constants pool throughout, x16/x17 hold the base pointers throughout, x9/x10 are recomputed per row, x11/x12 are loop counters, x13/x14 hold γ/β bases (affine). No register has two simultaneous roles. Verify against existing arm64 ops: x9-x16 + x6 are caller-saved on AAPCS64 § 6.1.1 (no save/restore needed in a leaf function); x17 is also caller-saved (IP1 — see register plan note above).

```rust
        // the FIRST register operand is a write. This is approximate — refine
        // if false positives surface in legitimate patterns.

            .filter(|line| line.contains(&format!("    fmov    {s_token}"))

                        || line.contains(&format!("    fmul    {s_token}"))
```

```rust
// SPDX-License-Identifier: Apache-2.0

//! LayerNorm codegen — x86_64 SSE2 (M14).
//!
//! Per-row 3-pass: mean → variance + inv_std → normalize + optional affine.
//! Native `sqrtss` — no libm call. Leaf when no-affine; op-local
//! `pushq %r12`/`pushq %r13` save inside emit body when has_affine == true.
//! `compute_callee_saved` is NOT modified — function-level prologue
//! surface unchanged (M13 pre-Task-5 arm64 precedent for op-local
//! save/restore).
//!
//! Register plan (M14 spec §8, N=1..2 scope, finalized):
//!   %rax  = x_j (inner counter); also temp for byte-offset compute before counter use
//!   %r10  = bound scratch (clobbered every emit_imm32_to_r10 call)
//!   %r11  = x_i (outer counter)
//!   %r8   = x_in  (per-row input ptr;  recomputed per row) — free at N≤2
//!   %r9   = x_out (per-row output ptr; recomputed per row) — free at N≤2
//!   %rbx  = src_base (set ONCE; callee-saved + op-local push/pop)
//!   %r14  = dst_base (set ONCE; callee-saved + op-local push/pop)
//!   %r12  = x_gamma (γ base ptr) — affine only, callee-saved + op-local push/pop
//!   %r13  = x_beta  (β base ptr) — affine only, callee-saved + op-local push/pop
//!   %xmm0 = s_acc; reused for s_var at end of Pass 2
//!   %xmm1 = s_mean (live Pass 2 + Pass 3)
//!   %xmm2 = s_inv_d (1/D constant; live across outer loop)
//!   %xmm3 = s_eps (1e-5; live across outer loop)
//!   %xmm4 = s_one (1.0; live across outer loop)
//!   %xmm5 = s_inv_std (Q4 constraint — held through Pass 3)
//!   %xmm6 = s_t (per-element temp)
//!   %xmm7 = s_g (γ_j load — affine only)
//!   %xmm8 = s_b (β_j load — affine only; xmm8 caller-saved on SysV, no AAPCS-style concern)
//!
//! All scratch in non-INPUT_REGS scope at N=1..2 (the M14 fixture range).
//! Higher-N (N=3..4) not validated; spec §8.7 documents the deferral.
//!
//! Why %rax for the inner counter (not %r10): emit_imm32_to_r10 clobbers
//! %r10 every call, so %r10 cannot hold a long-lived counter. %rax is
//! caller-saved scratch (free for op use without save/restore) and is
//! NOT touched by emit_imm32_to_r10.
//!
//! Stack alignment invariant (M-future foot-gun): always-pushed %rbx +
//! %r14 add +16 bytes (pair preserves alignment), conditional %r12 + %r13
//! add +16 (pair preserves). `pushq %rbp` in function prologue is +8 → odd.
//! M14 fixtures never co-locate LayerNorm with Softmax (the only op
//! emitting `call expf@PLT`), so misalignment never reaches a `call` site.
//! If a future fixture combines them, add a one-time `subq $8, %rsp`
//! adjustment outside the inner body.

use crate::abi::AbiContext;
use crate::asm::emit_imm32_to_r10;
use crate::buffer::BufferLoc;
use compiler::ast::Span;
use profile_api::LowerError;

#[allow(clippy::too_many_arguments)]
pub fn emit_layernorm(
    abi: &AbiContext,
    b: u64,
    d: u64,
    model_idx: usize,
    layernorm_idx: usize,
    src_loc: BufferLoc,
    dst_loc: BufferLoc,
    gamma_offset: Option<usize>,
    beta_offset: Option<usize>,
    _node_span: Span,
) -> Result<String, LowerError> {
    let lid = format!("{model_idx}_{layernorm_idx}");
    let has_affine = gamma_offset.is_some();
    debug_assert_eq!(has_affine, beta_offset.is_some(),
                     "γ/β offsets must be both Some or both None");

    let mut s = String::new();
    s.push_str(&format!(
        "    # layernorm (3-pass, {}affine): input [{b},{d}] -> output [{b},{d}]\n",
        if has_affine { "with " } else { "no-" },
    ));

    // Op-local callee-saved save. Push order: γ/β first if affine, then
    // base pointers. LIFO pop at function end mirrors this exactly.
    if has_affine {
        s.push_str("    pushq   %r12\n");
        s.push_str("    pushq   %r13\n");
    }
    s.push_str("    pushq   %rbx\n");
    s.push_str("    pushq   %r14\n");

    // Materialise base pointers ONCE before the outer loop.
    abi.materialise_ptr(src_loc, "%rbx", &mut s);
    abi.materialise_ptr(dst_loc, "%r14", &mut s);

    // Affine: materialise γ/β base pointers from params_reg into %r12/%r13.
    if has_affine {
        let g_off = gamma_offset.unwrap();
        let b_off = beta_offset.unwrap();
        let params_reg = abi.params_reg();
        if g_off == 0 {
            s.push_str(&format!("    movq    {}, %r12\n", params_reg));
        } else {
            s.push_str(&format!("    leaq    {}({}), %r12\n", g_off * 4, params_reg));
        }
        if b_off == 0 {
            s.push_str(&format!("    movq    {}, %r13\n", params_reg));
        } else {
            s.push_str(&format!("    leaq    {}({}), %r13\n", b_off * 4, params_reg));
        }
    }

    // Pre-loop hoisted constants — RIP-relative loads from .rodata pool
    // (emitted at end of this function).
    let inv_d_bits = (1.0_f32 / d as f32).to_bits();
    let eps_bits = 1e-5_f32.to_bits();
    let one_bits = 1.0_f32.to_bits();
    s.push_str(&format!("    # constants: inv_d=0x{:08x}, eps=0x{:08x}, one=0x{:08x}\n",
                        inv_d_bits, eps_bits, one_bits));
    s.push_str(&format!("    movss   .Lln_inv_d_{lid}(%rip), %xmm2\n"));
    s.push_str(&format!("    movss   .Lln_eps_{lid}(%rip), %xmm3\n"));
    s.push_str(&format!("    movss   .Lln_one_{lid}(%rip), %xmm4\n"));

    // === outer row loop: i in 0..b ===
    s.push_str("    xorq    %r11, %r11\n");
    s.push_str(&format!(".Lln_row_{lid}:\n"));
    s.push_str(&emit_imm32_to_r10(b as u32));
    s.push_str("    cmpq    %r10, %r11\n");
    s.push_str(&format!("    jge     .Lln_row_end_{lid}\n"));

    // Compute per-row pointers: %r8 = src_base + i*d*4; %r9 = dst_base + i*d*4.
    // Use %rax as transient byte-offset accumulator (then re-zeroed for Pass 1
    // counter use below — emit_imm32_to_r10 doesn't touch %rax).
    s.push_str(&emit_imm32_to_r10((d * 4) as u32));
    s.push_str("    movq    %r11, %rax\n");
    s.push_str("    imulq   %r10, %rax\n");          // %rax = i * d * 4 (byte offset)
    s.push_str("    leaq    (%rbx, %rax, 1), %r8\n");
    s.push_str("    leaq    (%r14, %rax, 1), %r9\n");

    // === Pass 1: μ = (1/D) · Σ x_j ===
    s.push_str("    xorps   %xmm0, %xmm0\n");        // s_acc = 0
    s.push_str("    xorq    %rax, %rax\n");          // x_j = 0 (counter)
    s.push_str(&format!(".Lln_p1_{lid}:\n"));
    s.push_str(&emit_imm32_to_r10(d as u32));        // bound = d → %r10
    s.push_str("    cmpq    %r10, %rax\n");
    s.push_str(&format!("    jge     .Lln_p1_end_{lid}\n"));
    s.push_str("    movss   (%r8, %rax, 4), %xmm6\n");
    s.push_str("    addss   %xmm6, %xmm0\n");
    s.push_str("    incq    %rax\n");
    s.push_str(&format!("    jmp     .Lln_p1_{lid}\n"));
    s.push_str(&format!(".Lln_p1_end_{lid}:\n"));
    s.push_str("    mulss   %xmm2, %xmm0\n");        // %xmm0 *= s_inv_d → s_mean (in %xmm0)
    s.push_str("    movss   %xmm0, %xmm1\n");        // copy s_mean to %xmm1 (stable through Pass 2/3)

    // === Pass 2: σ² = (1/D) · Σ (x_j − μ)²; inv_std = 1/sqrt(σ² + ε) ===
    s.push_str("    xorps   %xmm0, %xmm0\n");        // s_acc = 0
    s.push_str("    xorq    %rax, %rax\n");
    s.push_str(&format!(".Lln_p2_{lid}:\n"));
    s.push_str(&emit_imm32_to_r10(d as u32));
    s.push_str("    cmpq    %r10, %rax\n");
    s.push_str(&format!("    jge     .Lln_p2_end_{lid}\n"));
    s.push_str("    movss   (%r8, %rax, 4), %xmm6\n");
    s.push_str("    subss   %xmm1, %xmm6\n");        // %xmm6 -= mean
    s.push_str("    mulss   %xmm6, %xmm6\n");        // %xmm6 = (x − μ)²
    s.push_str("    addss   %xmm6, %xmm0\n");
    s.push_str("    incq    %rax\n");
    s.push_str(&format!("    jmp     .Lln_p2_{lid}\n"));
    s.push_str(&format!(".Lln_p2_end_{lid}:\n"));
    s.push_str("    mulss   %xmm2, %xmm0\n");        // s_var = s_acc * s_inv_d
    s.push_str("    addss   %xmm3, %xmm0\n");        // s_var += s_eps
    s.push_str("    sqrtss  %xmm0, %xmm0\n");        // s_var = sqrt(σ² + ε)
    s.push_str("    movss   %xmm4, %xmm5\n");        // %xmm5 = 1.0 (copy of s_one)
    s.push_str("    divss   %xmm0, %xmm5\n");        // s_inv_std = s_one / s_var
    //         ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^ Q4 constraint: ONE divss per row,
    //         OUTSIDE Pass 3. Pass 3 hot loop uses mulss %xmm5, ...

    // === Pass 3: y_j = (x_j − μ) · inv_std [· γ_j + β_j] ===
    s.push_str("    xorq    %rax, %rax\n");
    s.push_str(&format!(".Lln_p3_{lid}:\n"));
    s.push_str(&emit_imm32_to_r10(d as u32));
    s.push_str("    cmpq    %r10, %rax\n");
    s.push_str(&format!("    jge     .Lln_p3_end_{lid}\n"));
    s.push_str("    movss   (%r8, %rax, 4), %xmm6\n");
    s.push_str("    subss   %xmm1, %xmm6\n");        // %xmm6 -= mean
    s.push_str("    mulss   %xmm5, %xmm6\n");        // %xmm6 *= s_inv_std (NOT divss — Q4)
    if has_affine {
        s.push_str("    movss   (%r12, %rax, 4), %xmm7\n");  // γ_j → %xmm7
        s.push_str("    mulss   %xmm7, %xmm6\n");
        s.push_str("    movss   (%r13, %rax, 4), %xmm8\n");  // β_j → %xmm8
        s.push_str("    addss   %xmm8, %xmm6\n");
    }
    s.push_str("    movss   %xmm6, (%r9, %rax, 4)\n");
    s.push_str("    incq    %rax\n");
    s.push_str(&format!("    jmp     .Lln_p3_{lid}\n"));
    s.push_str(&format!(".Lln_p3_end_{lid}:\n"));

    s.push_str("    incq    %r11\n");
    s.push_str(&format!("    jmp     .Lln_row_{lid}\n"));
    s.push_str(&format!(".Lln_row_end_{lid}:\n"));

    // Op-local restores — strict LIFO of the entry pushes.
    s.push_str("    popq    %r14\n");
    s.push_str("    popq    %rbx\n");
    if has_affine {
        s.push_str("    popq    %r13\n");
        s.push_str("    popq    %r12\n");
    }

    // Per-function .rodata constants pool. Section directive switch must
    // be reconciled with asm.rs frame writer — see implementation notes
    // after this code block.
    s.push_str(".section .rodata\n");
    s.push_str(".align 4\n");
    s.push_str(&format!(".Lln_inv_d_{lid}: .long 0x{:08x}\n", inv_d_bits));
    s.push_str(&format!(".Lln_eps_{lid}:   .long 0x{:08x}\n", eps_bits));
    s.push_str(&format!(".Lln_one_{lid}:   .long 0x{:08x}\n", one_bits));
    s.push_str(".text\n");

    Ok(s)
}
```

**Implementation notes for the engineer:**

1. **Constant pool emission integration with `asm.rs`.** The `.section .rodata` / `.text` switch around the constant pool above assumes the per-function constant pool is emitted inline at the end of the function body. The exact emission mechanism depends on what `asm.rs::format_function_epilogue` (or a sibling helper) currently supports:

   - **If a per-function pool helper exists** in `profiles/x86_64/src/asm.rs`: use it. Pass the three constants to the helper at the end of `emit_layernorm`.

   - **If only inline `.text` content is currently supported**: the section-switch approach above works for x86_64 GAS syntax — the assembler emits the .rodata block then switches back to .text for any subsequent function. Verify by running `cargo run -p nflc -- compile <fixture> --profile x86_64` and inspecting the produced .s file. If the switch causes assembler errors or relocation issues, fall back to immediate materialization:

     ```rust
     // Inline materialization fallback:
     // movabsq $imm, %rax; movd %rax, %xmm2  — for each constant.
     // Costs 2 instructions per constant × 3 constants = 6 instructions hoisted
     // before the outer loop. Negligible per-function overhead.
     // (movabsq accepts 64-bit immediates; the upper 32 bits should be zero
     //  for f32 bit patterns. Use movabsq $0x{:016x}, %rax with the f32
     //  bits zero-extended to 64 bits, then movd %rax, %xmm — correct fp32
     //  bit pattern lands in the low half of the xmm register.)
     ```

2. **Confirm %rax is safe as inner counter.** Verify by reading `profiles/x86_64/src/asm.rs::emit_imm32_to_r10` — the helper should emit `movq $imm, %r10` (or similar) which writes only %r10, not %rax. If that helper happens to clobber %rax (unlikely), substitute another caller-saved scratch (e.g. %rdi at N=1 where it holds the input pointer that we've already materialised to %rbx and don't need anymore — but watch out for N≥2 where %rdi is INPUT_REGS[0] and downstream ops might re-read it).

3. **Stack alignment at `call` sites.** Module doc-comment notes the alignment invariant. M14 fixtures don't trigger it (no LayerNorm + Softmax co-location). If the M-future combination arises, the fix is a one-time `subq $8, %rsp` / `addq $8, %rsp` outside the inner body — exact placement verified by counting cumulative pushq depth at the `call expf@PLT` site in the combined-op asm.

```rust
/// Reference layernorm — bit-exact match with emitter at all D.
///
/// CRITICAL: uses sequential `for` reduction with `+=`. Do NOT replace
/// with `.iter().sum::<f32>()` — under -O3 LLVM may auto-vectorize
/// the sum into a SIMD tree-reduction, changing the order of float
/// additions and breaking bit-exact equivalence with the scalar 3-pass
/// asm. Sequential `for` keeps reduction strictly left-to-right; LLVM
/// does NOT reorder f32 adds without `-ffast-math` (Rust does not
/// enable it).
///
/// Uses pre-computed `inv_d = 1.0/d` and multiplication (`sum * inv_d`)
/// to match the emitter's compile-time `1.0/D` constant in `.rodata` —
/// `sum / d as f32` would diverge by 1 ULP for non-power-of-2 D.
pub fn layernorm_ref(
    input: &[f32],
    shape: &[usize],
    gamma: Option<&[f32]>,
    beta: Option<&[f32]>,
) -> Vec<f32> {
    let d = *shape.last().unwrap();
    let n = shape.iter().take(shape.len() - 1).product::<usize>();
    let inv_d = 1.0_f32 / d as f32;
    let eps = 1e-5_f32;

    let mut out = Vec::with_capacity(input.len());
    for r in 0..n {
        let row = &input[r * d..(r + 1) * d];

        let mut sum = 0.0_f32;
        for &x in row { sum += x; }
        let mean = sum * inv_d;

        let mut sumsq = 0.0_f32;
        for &x in row { sumsq += (x - mean) * (x - mean); }
        let var = sumsq * inv_d;
        let inv_std = 1.0_f32 / (var + eps).sqrt();

        for (i, &x) in row.iter().enumerate() {
            let normalized = (x - mean) * inv_std;
            let val = match (gamma, beta) {
                (Some(g), Some(b)) => normalized * g[i] + b[i],
                _ => normalized,
            };
            out.push(val);
        }
    }
    out
}
```
